上传Github

这一步我出现了err: Error: Spawn failed.

原因：

推测为为了上GitHub挂了代理，代理影响。

解决方法：

尝试了网上的各种方法，Linux使用rm -rf语句删除git提交内容文件夹（失败，系统不匹配），Windows使用rmdir语句（失败，提示不是内部或外部命令，也不是可运行的程序，或批处理文件）。

最后通过断开重连或更换代理的方式解决。简而言之，坐等。
